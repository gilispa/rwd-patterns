# Responsive Web Design Patterns

This is a **personal practice repository** focused on common
**Responsive Web Design (RWD)** layout patterns implemented using
**CSS Grid** and **Flexbox**.

---

## Patterns

- Two Columns
- Three Columns
- Column Drop
- Sidebar
- Layout Shifter
- Mostly Fluid
- Tiny Tweaks

Each folder contains a simple example of the corresponding layout pattern.

---

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
